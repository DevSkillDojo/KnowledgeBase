.. _resource_types:

Resource Types
==============

Learning resources come in many different forms. On this page we define a standardised catgorisation scheme
for learning types. This scheme will then be used as part of the learning paths.

+------+-------------------------------------------+
| Code |                                           |
+======+===========================================+
| B    | Book                                      |
+------+-------------------------------------------+
| WT   | Written Tutorial                          |
+------+-------------------------------------------+
| VTO  | Video Tutorial Only                       |
+------+-------------------------------------------+
| VTWE | Video Tutorial with hand-on exercise(s)   |
+------+-------------------------------------------+
| IT   | Interactive tutorial with live test cases |
+------+-------------------------------------------+
| OBC  | Online BootCamp                           |
+------+-------------------------------------------+
| PBC  | In person bootcamp                        |
+------+-------------------------------------------+
|      | Long form online course                   |
+------+-------------------------------------------+
|      | In person instructor-led course           |
+------+-------------------------------------------+
|      | Exam/interview prep                       |
+------+-------------------------------------------+
|      | Online skills assessment                  |
+------+-------------------------------------------+
|      | Formal industry certification             |
+------+-------------------------------------------+
|      | Code Example                              |
+------+-------------------------------------------+
|      | Reference Implementation                  |
+------+-------------------------------------------+
|      | Recorded Webinar                          |
+------+-------------------------------------------+
|      | Recorded Conference Talk                  |
+------+-------------------------------------------+
|      |                                           |
+------+-------------------------------------------+
|      |                                           |
+------+-------------------------------------------+
